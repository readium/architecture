DOM Storage

For security reasons, each publication (EPUB file) should be its own domain. Please set an item in localStorage from this page in order to see if you can access them in the touchHandling.epub file.

This item is using a rth-control- prefix. You shouldn’t be able to retrieve it from any other file. Alternatively, if you can get items with a rth-local- prefix or any other item, it means the document’s origin is the entire app, which raises security issues.
